Official Weather plugin for StreamController

Support for StreamDeck Plus
 - Three page widget
   - Current Weather
   - 5 Day forecast
   - hourly forecast
  <img width="226" height="126" alt="Kooha-2026-06-23-09-14-47" src="https://github.com/user-attachments/assets/fc93de69-b26e-4a53-ac64-131f8857fe9a" />
  
   - Automatic page cycle with 15, 30, 45, minute, or one hour interval  
 - Weather update interval, 30 minutes, or one hour refresh option
 - provider options:
   - Open-Meteo
   - OpenWeatherMap
   - Weather Underground
   - Weather.com
 - Dynamic backgrounds based on time of day, and weather conditions
 - Icon theme (coming soon)
 
Default icons provided by https://icons8.com/
